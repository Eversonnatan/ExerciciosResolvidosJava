4. Faça um programa que receba o preço de um produto, 
calcule e mostre o novo preço, sabendo-se que este sofreu um 
desconto de 10%. Por exemplo, se o produto custa R$ 100,00, 
o novo preço será 100 menos10% de cem, que é 100 x 10 / 100. 
Essa conta é igual a 10, 
ficando o novo preço R$ 90,00.


public static void main(String[] args) {
	double preco_prod ,preco_novo ;
	System.out.print("Digite o Preço do produto comprado: ");
	preco_prod = Double.parseDouble(System.console().readLine());
	preco_novo = preco_prod-((preco_prod*10)/100 );
System.out.printf("O preço do produto com 10% de desconto fica :",preco_novo);


    }
    
}
